Profile v1
Activated users manage their account — name, contact identity, language, theme, sign-out, and deletion — without contacting support.
The problem
Activated users have no self-serve way to update their account. Name edits, adding a missing sign-in method, language and theme switching, signing out, and deletion all currently route through support. Profile v1 brings these into the app as a single screen with branching flows. The hardest moment is account deletion — it must feel deliberate and irreversible without being theatrical.

Brand name MoneyMore stays in Latin script in both languages — never transliterated to Arabic.
Accessibility — what changes what you draw
  • Touch targets — 44×44px minimum on every interactive row, the avatar, theme segments, language radio rows, and dialog buttons. Profile rows look compact but the tap target must extend to the full row height.
  • Contrast — 4.5:1 minimum for text, with extra care on the destructive Delete account row. Red text on cream must meet AA in both light and dark themes.
  • Never colour-only for state. The Delete account row pairs red text with the trash icon. Disabled Save and Delete buttons need the disabled visual treatment plus reduced opacity, not just colour change.
  • Focus order — top to bottom. On the Profile root: avatar → Account section → Preferences → Help → Account actions. On Arabic the visual flips RTL but focus order stays top-to-bottom.
  • Avatar a11y label reads "Profile photo, initials shown" / "صورة الحساب، الحروف الأولى ظاهرة". Confirm with reviewer that this reads cleanly in both languages.
